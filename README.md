# Math Assistant GPT

This project presents a Smart Math Assistant developed using TinyLlama, LM Studio, Docker, and AnythingLLM, with a final website that includes a reliable quadratic solver.

## Technologies Used
- TinyLlama-1.1B
- LM Studio
- Docker
- AnythingLLM
- llama.cpp
- Python 3.10
- HTML, CSS, JavaScript

## Features
- Local custom GPT deployment
- Basic arithmetic responses through AnythingLLM
- Functional website with quadratic solver
- Project documentation and testing results

## Notes
The TinyLlama custom model deployed successfully but showed limited reliability for quadratic-equation solving. To ensure a correct final website, a deterministic quadratic solver was added.
