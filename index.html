Math Assistant GPT

A custom AI assistant for solving quadratic equations using TinyLlama, Docker, LM Studio, and AnythingLLM.

About Domain How to Use Development Testing

About the Project

This project presents a custom GPT-style math assistant built for quadratic equation solving. The assistant was created from a fine-tuned TinyLlama model trained on math examples and deployed locally using LM Studio and AnythingLLM.

The purpose of the system is to provide an interactive AI-based math support tool that can answer basic math questions and attempt step-by-step solutions for quadratic equations.

Chosen Domain

The chosen domain is Mathematics Education, with a focus on quadratic equations. This domain was selected because students often need guided support when solving algebraic expressions and equations.

  • Basic arithmetic support
  • Quadratic equation solving
  • Structured prompting using symbolic input
  • Interactive question-answer interface

How to Use the System

To use the math assistant effectively:

  1. Open the chat interface in AnythingLLM.
  2. Enter your question or equation.
  3. For best results, use quadratic equations in this format:

Eq(x**2 - 5*x + 6, 0)

Example Prompt:
Solve step by step quadratic equation: Eq(x**2 - 5*x + 6, 0)

The assistant will then attempt to provide a structured response. For basic arithmetic, simple direct questions such as 5 + 5 can also be used.

Development Process

The project development process included:

  • Fine-tuning a TinyLlama model using QLoRA on mathematical examples
  • Saving training output into adapter and merged folders
  • Converting the merged model to GGUF format using llama.cpp
  • Running the GGUF model in LM Studio
  • Connecting AnythingLLM to LM Studio through Docker

Challenges Faced

  • Docker installation and startup issues
  • Model conversion issues due to Python version compatibility
  • Prompt formatting mismatches
  • Incorrect quadratic equation outputs from TinyLlama

Solutions Implemented

  • Installed Python 3.10 for compatibility
  • Converted merged model to GGUF format
  • Connected LM Studio local server to AnythingLLM
  • Adjusted temperature and prompt structure for better consistency
